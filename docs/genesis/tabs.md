# Genesis - Tabs
File: `config/genesis/tabs.json`.  
Vanilla tabs: buildingBlocks, decorations, redstone, transportation, hotbar, search, misc, food, tools, combat, brewing, inventory.  
## Example
```json
[
  {
    "id": "misca.blocks",
    "title": "Misca-Blocks",
    "item": "brown_mushroom"
  },
  {
    "id": "misca.items",
    "title": "Misca-Items",
    "item": "bowl"
  }
]
```
